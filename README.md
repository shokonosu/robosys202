# plusコマンド(加筆済み)
標準入力から読み込んだ数字の加法・減法・乗法を行う。
* [plus](https://github.com/shokonosu/robosys202/blob/main/plus)
* 使い方はseq 5 | ./plusとコマンドを打つ。
```
$ seq 5 | ./plus
15 120 -15
```
