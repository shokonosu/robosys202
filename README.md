# plusコマンド
標準入力から読み込んだ数字を足す。
*・plus
*使い方はseq 5 | ./plusとコマンドを打つ。

$ seq 5 | ./plus
15

# calculateコマンド
*優先順位による計算プログラムと6の階乗を求める
*・calculate
*使い方はchmod +x calculateを入力したうえで、./calculateとコマンドを打つ

*$ ./calculate
*56 1.0 720  #計算結果
